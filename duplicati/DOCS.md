# Home Assistant Add-on: Duplicati

This add-on provides Duplicati for Home Assistant.

## Installation

The installation of this add-on is pretty straightforward and not different in
comparison to installing any other Home Assistant add-on.

1. Add the repository to your configuration.
1. Search for the "Duplicati" add-on in the Supervisor add-on store and install it.
1. Start the "Duplicati" add-on.
1. Open duplicati web interface via ingress
